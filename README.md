# 🚀 DSA Pattern-Wise LeetCode Solutions

This repository contains **Data Structures & Algorithms (DSA) patterns** along with explanations and implementations inspired by LeetCode problems.

The goal of this project is to help in:
- 📚 Understanding core DSA patterns
- ⚡ Improving problem-solving skills
- 🎯 Preparing for coding interviews
- 🧠 Recognizing patterns instead of memorizing solutions

---

## 📂 Project Structure

Each folder represents a **DSA pattern** with its own explanation and examples:

- 🔁 Two Pointer
- 🪟 Sliding Window
- ⚡ Fast & Slow Pointer
- 📈 Kadane’s Algorithm
- 📘 Prefix Sum
- 🧩 Merge Intervals
- 📚 Stack
- 🗂️ HashMap

Each folder contains:
- 📖 Concept explanation
- 🧠 Intuition
- ⚙️ Approach
- ⏱️ Complexity
- 💻 Code examples (Java)

---

## 🧩 Why Pattern-Based Learning?

Instead of solving problems randomly, learning by patterns helps you:
- Identify **similar problem types**
- Reduce thinking time in interviews
- Apply optimized approaches quickly
