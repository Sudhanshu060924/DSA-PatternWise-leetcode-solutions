# ⚡ Fast & Slow Pointer (Floyd’s Cycle)

Two pointers move at different speeds.

## 📌 When to Use
- Linked lists
- Cycle detection
- Middle element

## 🧠 Basic Idea
- `slow` → 1 step
- `fast` → 2 steps
- If they meet → cycle exists

## 🚀 Applications
- Detect cycle
- Find middle
- Happy number

## ⏱️ Complexity
- Time: **O(n)**
- Space: **O(1)**
