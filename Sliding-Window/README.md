# 🪟 Sliding Window Technique (DSA)

Used to process **subarrays/substrings efficiently**.

## 📌 When to Use
- Contiguous elements
- Subarray problems
- Max/min/count queries

## 🧠 Basic Idea
1. Use `left` and `right`
2. Expand window (`right++`)
3. Update result
4. Shrink window (`left++`)

## 🚀 Applications
- Max sum subarray
- Longest substring
- Minimum window substring

## ⏱️ Complexity
- Time: **O(n)**
- Space: **O(1) / O(k)**
