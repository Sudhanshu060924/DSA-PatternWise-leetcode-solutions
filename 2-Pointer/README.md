# 🔁 Two Pointer Technique (DSA)

The **Two Pointer Technique** is an efficient approach used in arrays and strings where two indices move through the data structure to solve problems in linear time.

## 📌 When to Use
- Arrays or strings
- Sorted arrays
- Pair or subarray problems
- Optimize O(n²) → O(n)

## 🧠 Basic Idea
1. Initialize two pointers (`left = 0`, `right = n - 1`)
2. Compare elements
3. Move pointers:
   - `left++`
   - `right--`
4. Stop when condition met

## 🚀 Applications
- Pair with given sum
- Remove duplicates
- Reverse array/string
- Palindrome check

## ⏱️ Complexity
- Time: **O(n)**
- Space: **O(1)**
